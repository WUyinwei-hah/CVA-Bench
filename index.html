CVA-Bench

Context-first benchmark pages for computer-use agent security

This site hosts the static CVA-Bench pages used to study whether agents comply with visibly suspicious popups when those popups appear tightly coupled to the current task.

Mainline families

Shopping, provider integration, payroll, password manager, admin console, and e-sign flows.

Variants

Clean, attack, and legit variants are injected by the runner instead of being exposed in the URL.

Public hosting

These pages are served through GitHub Pages so evaluation no longer depends on a localhost domain.

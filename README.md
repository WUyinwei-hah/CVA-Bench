# CVA-Bench Pages

This repository is the public GitHub Pages deployment repository for CVA-Bench.

It is intentionally kept small:

- static benchmark pages under `sites/`
- GitHub Pages deployment workflow under `.github/workflows/`
- public-facing benchmark notes under `docs/`

What is intentionally not included here:

- runners
- scorers
- scenario JSON files
- timing matrix configs
- private development utilities
- local experiment artifacts

The public site is served from:
